# Simon Memory Game (jQuery)

A simple Simon Says–style memory game built with JavaScript and jQuery.  
Press any key to start, watch the sequence, then repeat it by clicking the boxes in the same order. Each level adds one new step to the pattern.

## Features
- Random sequence generation (1–4)
- Level counter displayed in the heading
- Animated box flashes for the sequence playback
- Game over state + restart on keypress
- Click-based input validation against the generated sequence

## Tech Stack
- HTML / CSS
- JavaScript
- jQuery
